Alien

Alien - Trendy Multipurpose Template

Introduction

Alien is multipurpose HTML5 template that comes jam packed with 160+ pages and 22 + Home page variations. Alien comes with all the necessary building blocks for your site so that you don’t have to worry about designing a beautiful site. It will barely take some time to build and present beautiful websites of your choice from these pre-made components and building blocks.

Alien is a perfect choice for creating sites under General, Corporate, Creative, Portfolio, photography, Agency, Magazine, Restaurant, Hotel, Spa, Renovation, Fashion, E-commerce, Blog , Parallax, App Landing and Event category. Alien is fully responsive and built on top of the popular bootstrap framework.
