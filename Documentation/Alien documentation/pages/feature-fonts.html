Fonts

To set your custom google font go to Google Fonts then pick your favorite font(s) and then copy the embed link and finally replace with the already set google font.


<!DOCTYPE html>
<html>
    <head>
        ...

        Page Title - Alien

        
        
        
        ...
        
    </head>
    <body>
    ...
    </body>
</html>

Working with sass

Then, go to your sass file assets/build/sass/alien/variable.sass, and change your font-family name "Open Sans"

$family-sans-serif: "Open Sans", "Segoe UI", Roboto, Oxygen, Ubuntu, Cantarell, "Helvetica Neue", sans-serif !default

Working with css

Go to css/alien.css and replace font-family name "Open Sans" with yours.

body {
    overflow-x: hidden;

    letter-spacing: 0;

    color: #000;

    font-family: "Open Sans", "Segoe UI", Roboto, Oxygen, Ubuntu, Cantarell, "Helvetica Neue", sans-serif;
    font-size: 1rem;
    font-weight: 400;
    line-height: 1.4;
}
