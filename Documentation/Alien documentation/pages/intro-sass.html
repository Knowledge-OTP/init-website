Sass Structure

Sass is a CSS pre-processor, the most mature, stable, and powerful professional grade CSS extension language in the world; adding features that allow variables, mixins, functions. Bootkit supports Sass.

Sass Files are found under assets/build/sass

sass
│
├── alien
│   ├── components/
│   ├── mixins/
│   └── *.scss
│
└── alien.sass

CSS Naming Convention

For underlying CSS in Alien we follow Harry Roberts' style of block__element--modifier naming convention - or BEM.

Find out more about BEM at bem.info.
