Contact

There are 3 contact form variations contact-us-1.html, contact-us-2.html and contact-us-3.html. It's really really simple to setup contact form. You'll find contact form configurations in mailer/config.php file. Open that config file in your favorite editor and change the values accoding to sample code given below.


return array(
    /**
     * Email settings
     */
    'mailer' => array(
        'subject'    => 'Website Contact Form', // Email subject
        'from_email' => 'from@example.com', // Dynamic address cause problem, so use any of your email address
        'from_name'  => 'Jane Doe', // Sender name
        'to_email'   => 'to@example.com', // Recipient email address
        'to_name'    => '' // Optional
        ),

    /**
     * Fields map
     */
    'fields_map' => array(

        /**
         * Default contact form fields
         */
        'DEFAULT' => array(
            'name'    => 'Name',
            'email'   => 'Email',
            'message' => 'Message',
            ),

        /**
         * Contact form fields with phone number field
         * FORM2 is its id
         */
        'FORM2' => array(
            'name'    => 'Name',
            'email'   => 'Email',
            'message' => 'Message',
            'phone'   => 'Phone Number'
            ),
        )
    );
